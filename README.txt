### EN ###
This is a C# form app for doing MYSQL CRUD Operations.
There is students table and fields like name & student number in database.
There are Turkish words in Source Code so if you interest with this app please contact me for translating.
Database is located in sql7.freesqldatabase.com server.
Database is created with MySQL.
With this app you can show rows, add rows or delete rows.
With this app you can do both searching and listing with a criteria.
With this app you can Truncate Table.

Source code can be found in DatabaseApp folder or can be imported as a visual studio project with .sln file.
Executable file location: Executable\DatabaseApp.exe
Server password can be found in source code and database can be edited in www.phpmyadmin.co address.
Server may not work due its free. If necessary backup.sql file can be used in localhost.

Feel free to report the bugs if there.
